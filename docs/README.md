This folder will contain all docs folders for each client library.

Each folder will point to a Jekyll rendered Github page based on the Markdown docs generated by Swagger for each client library.
Each folder will be a gitsubmodule.

Ex. 
Python -> https://github.com/AlgolLLC/BinancePython

and then the corresponding Github page will be https://AlgolLLC.github.io/BinancePython
